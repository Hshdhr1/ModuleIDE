# Heroku AI Chat — Плагин для ExteraGram

[![Platform](https://img.shields.io/badge/platform-Android-green.svg)](https://developer.android.com/)
[![Language](https://img.shields.io/badge/language-Java-orange.svg)](https://docs.oracle.com/javase/)
[![Design](https://img.shields.io/badge/design-Material%203-blue.svg)](https://m3.material.io/)
[![License](https://img.shields.io/badge/license-MIT-lightgrey.svg)](LICENSE)

**Heroku** — это современный плагин искусственного интеллекта для клиента ExteraGram, предоставляющий удобный доступ к различным AI-провайдерам прямо из вашего Telegram.

## ✨ Возможности

### 🤖 Поддерживаемые провайдеры
- **Google Gemini** — передовая модель от Google с потоковым режимом
- **OpenRouter** — единый API к множеству моделей (GPT-4, Claude, Llama и др.)
- **OnlySQ** — российский AI-провайдер с поддержкой продвинутых моделей (DeepSeek-R1, Qwen) и режимом рассуждений
- **Любые OpenAI-совместимые API** — LocalAI, Ollama, vLLM и другие

### 💬 Чат с ИИ
- Красивый современный интерфейс в стиле Material Design 3
- Потоковые ответы с анимацией появления текста
- Сохранение истории чата локально на устройстве
- Экспорт истории в буфер обмена
- Контекстное меню «Спросить у Heroku» в Telegram-чатах

### ⚙️ Настройки
- Редактирование системного промпта с готовыми пресетами:
  - Полезный ассистент
  - Эксперт-программист
  - Креативный писатель
  - Психолог
  - Учитель
  - Саркастичный помощник
- Управление API ключами для каждого провайдера
- Загрузка списка доступных моделей
- Настройка температуры генерации (0.0–2.0)
- Включение/отключение потокового режима
- Включение/отключение подсветки синтаксиса
- Настройка уровня рассуждений для OnlySQ (minimal, low, medium, high)

### 🎨 Интерфейс
- Полностью на русском языке
- Material Design 3 (Material You)
- Адаптивная тёмная/светлая тема
- Плавные анимации переходов
- Индикаторы загрузки и состояния
- Обработка ошибок с понятными сообщениями

### 📝 Подсветка синтаксиса

- Inline код `code`

## 🔧 Установка

### Требования
- ExteraGram версии 1.0.0 или выше
- Android 8.0 (API 26) или выше
- OkHttp 4.x + OkHttp SSE
- Material Components for Android

### Шаги установки

1. **Скомпилируйте плагин:**
   ```bash
   ./gradlew assembleDebug
   ```

2. **Установите APK:**
   - Скопируйте `app-debug.apk` в устройство
   - Установите через файловый менеджер или ADB:
     ```bash
     adb install app-debug.apk
     ```

3. **Активируйте в ExteraGram:**
   - Откройте ExteraGram
   - Перейдите в Настройки → Плагины
   - Найдите «Heroku AI Chat» и включите его

4. **Настройте провайдеров:**
   - Откройте меню настроек Heroku
   - Добавьте API ключи для выбранных провайдеров
   - Выберите активную модель

## 📱 Использование

### Быстрый старт

1. Откройте ExteraGram
2. Нажмите на иконку ☰ (burger menu)
3. Выберите пункт **«Heroku»**
4. Начните общение с ИИ!

### Добавление провайдера

#### Google Gemini
1. Получите API ключ на [Google AI Studio](https://makersuite.google.com/app/apikey)
2. В настройках Heroku вставьте ключ в поле «Gemini API Key»
3. Нажмите «Загрузить список моделей»
4. Выберите нужную модель (рекомендуется `gemini-pro`)

#### OpenRouter
1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai/)
2. Создайте API ключ в личном кабинете
3. В настройках Heroku выберите провайдер «OpenRouter»
4. Вставьте API ключ и выберите модель

#### OnlySQ
1. Зарегистрируйтесь на [onlysq.ru](https://my.onlysq.ru/)
2. Получите персональный API ключ в профиле
3. Выберите провайдер «OnlySQ» в списке
4. Вставьте API ключ (начиная с patch 4.0.1 требуется Authorization header)
5. **Режим рассуждений**: OnlySQ поддерживает reasoning для моделей типа DeepSeek-R1
   - Уровни: `minimal`, `low`, `medium` (по умолчанию), `high`
   - Для сложных задач выбирайте `high` для подробного анализа
6. Популярные модели: `gpt-4o-mini`, `deepseek-r1`, `qwen-2.5`

**Пример использования режима рассуждений:**
```
Модель: deepseek-r1
Reasoning: high
Запрос: "Решите сложную логическую задачу..."
Ответ будет содержать цепочку рассуждений перед финальным решением.
```

#### Свой провайдер
1. Нажмите «Добавить провайдер»
2. Заполните поля:
   - Название (например, «Мой LocalAI»)
   - Endpoint (базовый URL API)
   - API Key (если требуется)
   - Модель по умолчанию
3. Нажмите «Тест соединения» для проверки
4. Сохраните

### Контекстное меню в чате

1. Долгое нажатие на сообщение в Telegram
2. Выберите «Спросить у Heroku»
3. Получите ответ от ИИ о выбранном сообщении

## 🔐 Безопасность

- API ключи хранятся в зашифрованном SharedPreferences
- Все сетевые запросы используют HTTPS
- Данные не передаются третьим сторонам
- Локальная история чата доступна только вам

## 🛠️ Разработка

### Структура классов

| Класс | Описание |
|-------|----------|
| `HerokuMainActivity` | UI чата, адаптер сообщений, обработка ввода |
| `SettingsManager` | CRUD операций для настроек и провайдеров |
| `CustomProvider` | Модель данных провайдера (JSON сериализация) |
| `NetworkService` | HTTP/SSE запросы к API провайдеров |
| `ChatManager` | Сохранение/загрузка истории, экспорт |

### Добавление нового провайдера

1. Добавьте пресет в `SettingsManager.create***Preset()`
2. Реализуйте методы отправки в `NetworkService` (если отличается от OpenAI)
3. Обновите UI настроек при необходимости
